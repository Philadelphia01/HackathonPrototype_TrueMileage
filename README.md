# TrueMileage

**TrueMileage** is an AI-powered mobile platform that supports used car buyers with real-time vehicle verification and mechanical diagnostics. It allows users to assess a car's condition and verify online listings using only a smartphone. 

## Features
- Vehicle identification via VIN
- AI-powered engine diagnostics
- Real-time vehicle condition report
- Transparent buying assistance

```bash
npm install
npm run dev
